## Project Overview
**Campus Connect** is a centralized digital portal developed specifically for the students of All Saints Catholic College. This platform serves as a unified hub for real-time school information, house competition tracking, and student engagement tools. The primary objective of this project was to utilize a professional framework and perform extensive custom development to meet the administrative and cultural requirements of a Catholic secondary school environment.

## Technical Stack
*   **Development Environment:** Sublime Text 4
*   **Languages:** HTML5, CSS3, JavaScript (jQuery, Skel.js)
*   **Version Control:** GitHub
*   **Base Framework:** [HTML5UP 'Hyperspace'](https://html5up.net) (Responsive Grid & Navigation)

## Key Features and Customizations
Significant modifications and original components were integrated into the base framework to create a unique and functional tool:

*   **Interactive House Standings:** Engineered a custom data visualization section to track points for Teresa, Aikenhead, Marum, Romero, Cahill, and Patrick houses using specific school branding.
*   **Gamification (Daily Wordle):** Integrated a functional Wordle-style game component to drive daily student engagement and house-point incentives.
*   **Advanced UI/UX:**
    *   Implemented a functional **Dark Mode** toggle for improved user accessibility.
    *   Applied a frosted-glass (**Glassmorphism**) overlay to the hero section to ensure title readability over background imagery.
*   **School Localization:**
    *   Integrated a formal **Acknowledgement of Country** to honor the Traditional Owners of the land.
    *   Developed a specialized **Contact Us** system featuring school-domain validation placeholders (`@sydstu.catholic.edu.au`).
    *   Restructured the sidebar navigation for school-specific utility, including Calendar, Resources, and News sections.
*   **Responsive Optimization:** Refined CSS media queries to ensure the House Standings and Wordle game components remain fully functional across desktop and mobile devices.
